## 🚀 핵심 기능
### 사용자의 위치를 기반으로 근처의 가까운 식당 5개를 추천합니다!
- 음식점 사진, 이름, 카테고리, 다른 사용자들이 클릭한 횟수, 나의 위치로부터의 거리와 같이 다양한 세부정보를 확인할 수 있습니다.
- <b>현재 영업중인 음식점</b>만 추천하므로 휴업일, 브레이킹타임에 방문하는 불상사를 예방할 수 있습니다.
<img width="800" alt="random" src="https://user-images.githubusercontent.com/28749734/133468515-25d9fa4b-5b04-485f-9c09-9d74d9e48c60.png">

<br> 

### 그래도 어렵다면 직접 골라드리겠습니다!
- 보여지는 5개의 음식점 중, 고르기가 힘들다면 단 하나의 음식점을 랜덤으로 선택합니다.
- 고르기 전, 포함하고 싶지 않은 음식점은 슬라이드하여 새로운 음식점으로 대체할 수 있습니다.
